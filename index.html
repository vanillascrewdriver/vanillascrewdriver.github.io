Neural Networks are a method of data processing that mimic the concept of neural activity in the brain to convert some input into an appropriate output by passing values through a system of nodes, called neurons. They were first introduced in the mid-20th century, but then phased out of interest due to their lack of practicality at the time. With modern processing, however, neural networks have come back into interest again, even making it into the news through applications like AlphaGo.

How they work[edit]

A neural network consists of a series of layers of nodes. There is the input layer, the hidden layers, and the output layer. When you use it, you put one value into each of the input nodes, and then run a program to push the values forward through the network, eventually filling each of the output nodes with a value. In the example of a basic hand-written digit recognition network, you might make each input node represent a pixel, with input being 0 or 1 (white or black), and have each of ten nodes represent a possible digit from 0 to 9.

But how does the neural network produce an output? The values of the nodes in the first layer are provided by the user, and then the network calculates the values for each of the nodes one layer at a time. In a simple neural network, to calculate the value of a node, you take each of the nodes in the previous layer, multiply their value by a weight corresponding to the pair of nodes, add all of those values up, and add a bias specific to the node you are calculating. By iteratively doing this for each layer, we then simplify the calculation of the entire network to just k matrix multiplications, where k is the number of layers (excluding the input layer). This does not change the number of mathematical operations we have to do, however, as each matrix multiplication consists of an equal number of operations as would doing it via the original equation. So why do we use linear algebra?

In recent advances in computing, the development of the GPU has led to a massive improvement in the speed of computing many linear algebra functions. This is because the GPU makes use of parallel processing, running the same process on multiple processors for different values at the same time. Since matrix multiplication is extremely repetitive in its arithmetic, the GPU is able to perform hundreds of these calculations at once, while with normal computing, each one would have to be calculated one at a time. This allows neural networks that make use of linear algebra to be computed multiple orders of magnitude faster than with the traditional method.

GPU's, or Graphics Processing Units, were originally intended to take advantage of parallel processing in processing graphics, which are heavily dependent on vector equations and linear algebra. Computer scientists realized they could take advantage of this in many areas of machine learning, sparking the sudden rise in popularity of neural networks and machine learning in recent years.

Application[edit]

Neural networks can be used for things as basic as digit recognition to as complicated as beating the world champion in the ancient game of Go. They are used by Google on a massive scale for image recognition, and by autonomous cars for environmental sensing. Recent advances have begun using them for forecasting as well as diagnosis assistance in hospitals. A more morally debatable use of neural networks has been use in the court system with Compas. As they become more popular, people find more and more uses for these versatile tools, and with growing speed of processing, they will only become more popular.
